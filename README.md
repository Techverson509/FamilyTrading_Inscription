# 📄 Fòmilè - PDF Otomatik

Fòmilè HTML pou ranpli done epi **telechaje PDF ofisyèl otomatikman**. Pa gen backend, mache dirèkteman nan navigatè.

## ✨ Kijan sa mache?

1. **Ranpli fòm nan** (non, telefòn, imèl...)
2. **Klike "📄 Telechaje PDF mwen"**
3. **PDF ranpli parèt otomatikman**
4. **Enprime oswa voye pa imèl/WhatsApp**
